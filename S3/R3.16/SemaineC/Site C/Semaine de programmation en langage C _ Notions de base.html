Notions de base

Cette unité comporte une série de petits exercices indépendants qui vous font aborder les notions de base avec peu de lignes de code. Nommez les programmes comme il est demandé et conservez bien tous les petits programmes sources (.c) que vous écrivez. Dans certains exercices, des conseils sont donnés pour vous aider à partir dans la bonne direction. Lisez les avant de commencer à programmer.

Rappels

Commande de compilation pour produire executable à partir de source.c

$ gcc -std=c99 -Wall -Wextra -g -o executable source.c

Code de hello-world.c

#include <stdio.h>
#include <stdlib.h>

int main(void)
{
	printf("Hello world!\n");

	return EXIT_SUCCESS;
}

Appel de fonction : Conversion Fahrenheit/Celsius

  1. Écrire le programme conversion.c qui affiche une table de conversion de températures en degrés Fahrenheit vers des températures en degrés Celsius.

Le programme donnera la conversion de 0 °F à 200 °F par pas de 20 °F.

Pour effectuer la conversion, la fonction main() appellera la fonction celsius() que vous définirez avec le prototype suivant  

float celsius(float fahrenheit)

et qui utilisera la formule c = 5 9 ⋅ ( f − 32 ) où c représente la température en degrés Celsius et f représente la température en degrés Fahrenheit.

Pour la mise en forme de la table, vous utiliserez le caractère de tabulation '\t'.

  1. Une fois le programme fonctionnel, créer une version (commit) qui enregistre votre travail.

Code fourni : readline()

Pour lire une chaîne de caractères qui contient des espaces, la fonction readline() est fournie. 

#define _POSIX_C_SOURCE 200809L  // getline in POSIX.1-2008

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

/*
 * readline() reads an entire line from stream and stores at most one less than
 * size characters into the buffer pointed to by buffer.
 * A terminating null byte ('\0') is stored after the last character in the
 * buffer.
 *
 * On success, readline() returns the number of characters read, excluding the
 * terminating null byte ('\0').
 *
 * The function returns -1 on failure to read a line (including end-of-file
 * condition).
 */
ssize_t readline(char *buffer, size_t size, FILE *stream)
{
	char *line = NULL;
	size_t len = 0;
	ssize_t nread;
	ssize_t ncopied = -1;

	nread = getline(&line, &len, stream);
	if (nread != -1) {
		// copy at most size - 1 bytes
		ncopied = nread < (ssize_t)size - 1 ? nread : (ssize_t)size - 1;
		memcpy(buffer, line, ncopied);
		// terminating null byte
		buffer[size] = '\0';
	}
	free(line);

	return ncopied;
}

Une fois intégrée à votre code, la fonction readline() s'utilise de la manière suivante : 

#define BUFFER_LEN 64

char buffer[BUFFER_LEN];
ssize_t nread;

nread = readline(buffer, BUFFER_LEN, stdin);

Conseils (avant de coder !)

  • Pour une hauteur h, combien de caractères c comporte la base ?
  • Pour un niveau n, combien de caractères c sont affichés sur une ligne ?
  • Pour un niveau n, combien d'espaces sont affichées avant le premier caractère ?
  1. Une fois le programme fonctionnel, créer une version (commit) qui enregistre votre travail.
  2. Modifier votre programme pour qu'il récupère la hauteur et le caractère à afficher dans les arguments passés en paramètres du programme. Le programme doit tester que le bon nombre d'arguments est donné.

Conseils

Les arguments "3" et "0" sont deux chaînes de caractères. Il faut donc les convertir :

  • "3" en entier 3 (à l'aide d'atoi()),

Conversion bin2dec() : base 2 vers base 10

  1. Créer un nouveau fichier source bin2dec.c.
  2. Écrire une procédure bin2dec() de prototype 
    void bin2dec(char bin[], char dec[])
    qui convertit une chaîne de caractères bin contenant les caractères '0' et '1' représentant un entier naturel exprimé en base 2 en une chaîne de caractères dec contenant les caractères '0' à '9' représentant cet entier naturel exprimé en base 10.
  3. Écrire un main() qui permet de tester votre procédure.

Conseils

Pour écrire le résultat de la conversion dans la chaîne de caractères dec, on utilisera la fonction sprintf() qui fonctionne comme printf() avec un argument supplémentaire pour indiquer dans quel tableau écrire le résultat.

  1. Une fois le programme fonctionnel, créer une version (commit) qui enregistre votre travail.

Conversion dec2bin() : base 10 vers base 2

  1. Créer un nouveau fichier source dec2bin.c.
  2. Écrire une procédure dec2bin() de prototype 
    void dec2bin(char dec[], char bin[])
    qui fait l'opération inverse de bin2dec() : elle convertit une représentation en base 10 contenue dans dec en une représentation en base 2 contenue dans bin.

Conseils

  • Convertir dec en entier naturel à l'aide d'atoi().
  • Utiliser les divisions successives par 2 (cf. cours d'architecture de première année).
  • Le reste de la division entière peut être obtenu avec l'opérateur modulo %.
